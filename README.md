Spooky’s Ticketing Platform

A web-based event ticketing platform built to handle ticket sales, user registration, and secure payments for events.

Overview

Spooky’s Ticketing Platform allows event organizers to sell tickets online while providing users with a simple and smooth checkout experience. The system handles ticket selection, user details, and payment processing, with a focus on reliability and clean user flows.

This project was built as a production-style web application with real payment integration and practical business logic.

Key Features
	•	Event ticket listing and purchase flow
	•	Secure payment integration using Paystack
	•	User-friendly checkout experience
	•	Automatic ticket generation after successful payment
	•	Admin-ready structure for managing events and tickets
	•	Responsive UI for desktop and mobile users

Tech Stack
	•	Backend: Django (Python)
	•	Frontend: HTML, CSS, JavaScript
	•	Payments: Paystack
	•	Database: PostgreSQL / SQLite

Project Focus
	•	Building a secure and reliable payment flow
	•	Designing clear user journeys from ticket selection to checkout
	•	Structuring backend logic to support real users and live transactions
	•	Ensuring the application can be extended for additional event features
